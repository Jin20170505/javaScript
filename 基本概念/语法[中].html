3. 变量

ECMAScript的变量是松散类型的，所谓的松散类型就是可以用来保存任何类型的数据。每个变量仅仅 是一个用于保存值的占位符而已。

  1. var message; ----- undefined
  2. var message = "hi";
  3. var message = "hi"; message = 100;有效，但不推荐这样。
  4. 省略var操作符，变量就成了全局变量。但是，不推荐使用。 这样做，不方便维护，代码混乱。在严格模式下，会抛出ReferenceError错误。
  5. 一条语句定义多个变量：var message="hi",found = false,age = 29;

4. 数据类型

ECMAScript中有5种简单数据类型：Undefined、Null、Boolean、Number和String。还有一种复杂的数据类型--Object，Object本质上是由一组无序的名值对组成。

4.1 typeof 操作符

用来检测给定变量的数据类型。

var message ="HI"; console.log(typeof(message);
  1. "undefined"-----未定义
  2. "boolean"-------布尔值
  3. "string"--------字符串
  4. "number"--------数值
  5. "object"--------对象或者null
  6. "function"------函数
